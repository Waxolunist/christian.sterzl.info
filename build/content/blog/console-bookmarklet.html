Console-Bookmarklet - A small debugging tool for mobile

I added some debugging code to this site to see what's going on on my iPhone. I blogged about it: How to install Bookmarklets in iOS Safari. But I didn't wanted to convolute my code with too much debugging code.

So I searched again for a good bookmarklet, which plays nice on mobiles, to see the output of console.log but I couldn't find any.

I tried Firebug Lite, but this was too heavy and felt cumbersome on my iPhone. Especially, when I turned my iPhone around, I couldn't see anything of my page because Firebug took the whole screen. So, this didn't fit my needs.

That's when I started to write my own. The main requirements were:

  • Take the output of console.log and display it in a div.
  • The div should be draggable, resizeable and closeable.

I decided to take the existing dialog JQuery UI Dialog and made it touch enabled with a touch library. The rest has been easy as I just had to take my existing debugging code and move it to my new bookmarklet.

I host the code of this small utility on github.
